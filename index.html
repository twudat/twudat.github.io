FREE-XXX-O-DUS Kodi Adult Addon

FREE-XXX-O-DUS is a Kodi addon that allows you to stream your favorite Nroppy Vids. With FREE-XXX-O-DUS, you can easily search for and watch your Nroppy videos. The addon is regularly updated with new site scrapers, so you'll never run out of things to watch.

Download the addon from here :repository.free-xxx-o-dus.zip

Instructions for download:

  1. Download the ZIP file from the repository.
  2. Open Kodi and go to "Add-ons".
  3. Select "Install from zip file".
  4. Navigate to the downloaded ZIP file and select it.
  5. Wait for the addon to install.

Instructions for Kodi:

  1. Go to the Filemanager in Kodi
  2. Click on Add Source
  3. In Add file source window, click on <None> .
  4. When the field is activated, type this url: https://twudat.github.io/
  5. Click on OK, under 'Enter a name for this media source',
    give it : repository.free-xxx-o-dus
  6. Click on OK to close the window. You are now done. To test if it works, go to the just added source in Filemanager, you should see the zipfile
  7. Now you can go to Addons, and install from Zip. You see the newly added source there.
