Snoddenrop Kodi Addon

Snoddenrop is a Kodi addon that allows you to stream your favorite Nroppy Vids. With Snoddenrop, you can easily search for and watch your Nroppy videos. The addon is regularly updated with new site scrapers, so you'll never run out of things to watch.

Download the addon from here repository.snoddanrop.zip.

Instructions:

  1. Download the ZIP file from the repository.
  2. Open Kodi and go to "Add-ons".
  3. Select "Install from zip file".
  4. Navigate to the downloaded ZIP file and select it.
  5. Wait for the addon to install.
